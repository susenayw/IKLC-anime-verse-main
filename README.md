## üíª Project Description

Animeverse adalah sebuah aplikasi **mobile** yang dirancang untuk membantu pengguna menjelajahi dan mengelola daftar anime secara **terstruktur** dan **interaktif**.

### üåü Fitur Utama

Aplikasi ini menawarkan beberapa fungsionalitas inti, antara lain:

* **Daftar Anime:** Menampilkan daftar anime yang komprehensif.
* **Informasi Detail:** Pengguna dapat melihat detail lengkap setiap anime, meliputi:
    * **Judul**
    * **Deskripsi Singkat**
    * **Rating**
    * **Sinopsis Lengkap**
* **Klasifikasi Genre:** Memungkinkan penyaringan dan penemuan anime berdasarkan **genre**.
* **Manajemen Favorit:** Pengguna dapat **menambahkan** anime ke dalam daftar favorit pribadi mereka.

### üîë Autentikasi Pengguna

Animeverse mendukung fitur login yang fleksibel:

1.  **Akun Lokal:** Login menggunakan **email dan password** yang dibuat oleh pengguna.
2.  **Google Sign-In:** Login cepat dan mudah menggunakan **akun Google** Anda.

> **Catatan:** Pengguna akun lokal dapat **mengubah password** mereka. Fitur perubahan password **tidak tersedia** untuk pengguna yang login menggunakan akun Google.

### ‚öôÔ∏è Sumber Data (API)

Aplikasi ini berjalan berkat integrasi dengan **Jikan API**, sebuah *unofficial MyAnimeList API*, yang menyediakan data-data anime yang lengkap dan *up-to-date*.

* **Jikan API Link:** [https://jikan.moe/](https://jikan.moe/)
